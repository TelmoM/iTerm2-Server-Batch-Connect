# iTerm2 Server Batch Connect

Script for iterm2 to open a new window with panes, one for each server.

### Instalation
```
pip3 install iterm2
pip3 install pyobjc
```

Copy the script(s) to the folder: `~/Library/ApplicationSupport/iTerm2/Scripts`
